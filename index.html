What is STORY HOOK?

Storybook

  • Storybook is the most popular UI component development ENVIRONMENT
  • It runs OUT OF APP, and makes possible to build components in ISOLATION, out of any parent component context
  • It makes possible to see different STATES of components
  • It may be used as a COMPONENT LIBRARY, for COMPONENT TESTING, and for SHARING COMPONENTS accross the teams as a STATIC WEBSITE on it's own

About Storybook

  • It was created by Sri Lankan startup called Kadira, as their side project
  • In 2016 Kadira shut down, and Storybook was almost dead
  • Then enthusiast Norbert de Langen stepped up, and since then Storybook is community-driven
  • It's being used by GitHub, Dropbox, Airbnb, Mozilla and maybe Esome
  • It supprots React, React Native, Vue, Angular, Svelte...Starships :)

How can we use it?

  • Set it up using: npx -p @storybook/cli sb init
  • Customize Storybook CONFIG file if needed to specify components folder, or setup addons
  • Write stories, where each story represents single component STATE, displayed in ISOLATION
  • Use Knobs and Actions to make component testing more flexible
  • Build a static website if needed using: npm run build-storybook
